AUTOKEEPER
SETUP & INSTALLATION GUIDE

Getting Started

AutoKeeper is a self-contained web app — no app store, no account, no subscription. Your data is stored directly on your device and never leaves it. Below are instructions for every platform.

FILES YOU RECEIVED
autokeeper.html
The entire app. Open this in any browser to run it.
sw.js
Service worker. Enables offline use and home screen installation.
manifest.json
PWA config. Tells your phone how to display the app icon and name.
instructions.html
This file. You can delete it once you're set up.
⚠️ Keep all three app files (autokeeper.html, sw.js, manifest.json) in the same folder. They need to be together to work correctly.
OPTION A — USE IT AS A LOCAL FILE (SIMPLEST)

No internet or server needed after the first open. Just open the file directly in your browser. Data saves automatically to that browser on that device.

🖥️
Windows / Mac / Linux
Chrome, Edge, Firefox, Safari
1
Download all 3 files into one folder
Keep autokeeper.html, sw.js, and manifest.json together.
2
Double-click autokeeper.html
It opens in your default browser. That's it — the app is running.
3
Bookmark it for easy access
Press Ctrl+D (Windows) or Cmd+D (Mac) to bookmark. Your data persists automatically in the browser.
✓ Data saves in your browser's localStorage automatically. As long as you open the same file in the same browser, everything is there.
📱
iPhone / iPad
Safari required for home screen install
1
Transfer the 3 files to your iPhone
AirDrop the folder from your Mac, or use iCloud Drive / Files app. All 3 files must stay in the same folder.
2
Open the Files app → find autokeeper.html
Tap it once — it will open in Safari.
3
Add to Home Screen
Tap the Share button (box with arrow pointing up) → scroll down → tap "Add to Home Screen" → tap Add.
4
Open from your home screen
The AutoKeeper icon now appears on your home screen. Tap it to open full-screen, like a native app. Always open it this way — not from Safari — to keep your data consistent.
⚠️ On iPhone, when opening a local file, data is stored per-session in that browser context. For reliable saving, use the hosted option below (Option B) which gives a permanent URL.
🤖
Android
Chrome recommended
1
Transfer the 3 files to your Android device
Use a USB cable, Google Drive, or Bluetooth. Place all 3 files in the same folder (e.g. Downloads).
2
Open autokeeper.html in Chrome
Use your file manager app, tap the file, choose "Open with Chrome".
3
Install to home screen
Tap Chrome's ⋮ menu → "Add to Home screen" → Add. The app icon appears on your home screen.
✓ Android Chrome handles local PWA installs well. Data saves reliably in localStorage.
OPTION B — HOST IT ONLINE (RECOMMENDED FOR PHONES)

Hosting gives you a permanent URL you can open from any device, any browser, any time. This is the most reliable way to use AutoKeeper on your phone — especially iPhone. All options below are free.

▲
Netlify Drop (Easiest — 60 seconds)
No account required
1
Go to app.netlify.com/drop on your computer
2
Drag your folder into the drop zone
Drag the entire folder containing all 3 files onto the page.
3
Get your URL
Netlify gives you a free URL like https://fancy-name-123.netlify.app. Open it on your phone, add to home screen.
✓ Free forever. No login needed. URL works on any device.
🐙
GitHub Pages (Free, Permanent)
Requires a free GitHub account
1
Create a free account at github.com
2
Create a new repository → upload all 3 files
Name the repo anything (e.g. autokeeper). Upload files via the web interface.
3
Enable GitHub Pages
Go to repo Settings → Pages → Source: main branch. Save. Your app is live at https://yourusername.github.io/autokeeper/autokeeper.html
ℹ️ GitHub Pages is permanent and free. Great option if you want to update the app later.
INSTALL TO IPHONE HOME SCREEN FROM A HOSTED URL
1
Open the URL in Safari
Must be Safari — Chrome on iPhone cannot install PWAs.
2

The box-with-arrow icon at the bottom of the screen.
3

AutoKeeper appears on your home screen with an icon. Opens full-screen, no browser chrome.

✓ After install, the app works fully offline. Your data never leaves your device.
FAQ
Will my data sync between my phone and computer?
Not automatically — data lives in each browser's localStorage separately. To move data between devices, use the CSV export button in the app, then import it manually. True sync would require a backend server, which is a future enhancement.
What happens if I clear my browser data?
Clearing browser data or cache will erase your AutoKeeper data. Export a CSV backup regularly, especially before clearing browser storage.
Can I share this with someone else?
Yes — just share the URL if hosted, or send them the 3 files. Their data is completely separate from yours.
Does it work without internet?
Yes, after the first load. The service worker caches all app files so it works offline. If you're using a local file (not hosted), it always works offline.
Can I update the app without losing my data?
Yes. Data is stored in localStorage independently of the app files. Replacing the HTML/JS files doesn't touch your saved vehicles and logs.
